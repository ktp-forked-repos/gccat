3.7.236. Subset sum

  • 𝚠𝚎𝚒𝚐𝚑𝚝𝚎𝚍_𝚙𝚊𝚛𝚝𝚒𝚊𝚕_𝚊𝚕𝚕𝚍𝚒𝚏𝚏.

Denotes the fact that, by reduction to subset sum, deciding whether a constraint has a solution or not was shown to be NP-hard. The subset sum problem can be described as follows: given a finite set of integers in ℤ + and an integer s in ℤ + , does any subset sum equal exactly s?
