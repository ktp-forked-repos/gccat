5.177. in

DESCRIPTIONLINKSGRAPHAUTOMATON
Origin

Domain definition.

Constraint

𝚒𝚗(𝚅𝙰𝚁,𝚅𝙰𝙻𝚄𝙴𝚂)

Synonyms

𝚍𝚘𝚖, 𝚒𝚗_𝚜𝚎𝚝, 𝚖𝚎𝚖𝚋𝚎𝚛.

Purpose

Enforce the domain variable 𝚅𝙰𝚁 to take a value within the values described by the 𝚅𝙰𝙻𝚄𝙴𝚂 collection.

Example
(3,1,3)

  • 𝚅𝙰𝚁 can be set to any value of 𝚅𝙰𝙻𝚄𝙴𝚂.𝚟𝚊𝚕.

  • One and the same constant can be added to 𝚅𝙰𝚁 as well as to the 𝚟𝚊𝚕 attribute of all items of 𝚅𝙰𝙻𝚄𝙴𝚂.

Remark

Entailment occurs immediately after posting this constraint.

The 𝚒𝚗 constraint is called 𝚍𝚘𝚖 in Gecode (http://www.gecode.org/), and 𝚖𝚎𝚖𝚋𝚎𝚛 in MiniZinc (http://www.minizinc.org/). In MiniZinc the 𝚟𝚊𝚕 attribute is not necessarily fixed, i.e. it can be a domain variable.

Signature
